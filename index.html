How to install in Xcode
1Download the .codesnippet file using the button on the right.
2Open Finder and press ⇧⌘G, then paste: ~/Library/Developer/Xcode/UserData/CodeSnippets/
3Drop the downloaded file into that folder.
4Restart Xcode (or use File → Close & Reopen). Your snippet now appears in the snippet library and autocomplete.
